# Facility Management System

A comprehensive web-based facility management platform built with Firebase that enables users to report facility issues with photos and allows admins to track, manage, and resolve issues efficiently.

## 🎯 Features

### User Features
- ✅ Report facility issues with photo uploads
- ✅ CRUD operations on feedback (Create, Read, Update, Delete)
- ✅ Review and track status of reported issues
- ✅ Real-time updates on issue resolution

### Admin Features
- ✅ View all reported issues
- ✅ Document issue resolution with photos and description
- ✅ Add date and time stamps to resolutions
- ✅ CRUD operations on issue status
- ✅ Generate reports on:
  - Pending issues
  - New issues (last 24 hours)
  - Unaddressed issues (past 14 days)
  - User reviews and ratings

## 🏗️ Tech Stack

- **Frontend:** React.js / Vue.js
- **Backend:** Firebase Realtime Database
- **Storage:** Firebase Cloud Storage
- **Authentication:** Firebase Authentication
- **Hosting:** Firebase Hosting (optional)
